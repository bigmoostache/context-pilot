Security & Privacy

Context Pilot is a local-first, open-source AI coding assistant. Your code never touches our servers — because we don't have any. This page documents our security architecture, data handling, and every third-party service the tool communicates with.

🏠

Local-First

Runs entirely on your machine. No cloud, no intermediary.

🔓

Open Source

MIT license. Every line auditable on GitHub.

🚫

Zero Telemetry

No analytics, no tracking, no phone-home.

🔑

Your Keys, Your Data

API keys stay local. Sent only to the provider they belong to.

Privacy Controls

Context Pilot's architecture makes data exfiltration structurally impossible. There is no Context Pilot cloud, no telemetry endpoint, no intermediary proxy.

No data collection

We never receive your code, conversations, memories, or any usage data. There is no server for it to go to.

Local storage only

All state — conversations, entity databases, memories, logs — is stored under .context-pilot/ on your filesystem. Nothing leaves the machine.

API key isolation

Your API keys live in a local .env file. Each key is sent only to its respective provider — never to us or any third party.

Open-source audit trail

The entire codebase is MIT-licensed. Every network call, file access, and tool invocation is auditable. Verify yourself: github.com/bigmoostache/context-pilot.

Offline capable

Context Pilot works without internet. Only LLM inference, web search, and web scraping require network — all user-initiated and user-configured.

Cryptographic integrity

Critical configuration files (lint rules, CI scripts, deploy scripts) are protected by a SHA-256 hash chain. Unauthorized modifications are detected automatically.

Data Handling

Every piece of data is either stored locally or sent to a provider you explicitly configure. Here's the complete breakdown.

Data Type Storage Transmitted?
Code context & prompts Assembled in-memory during sessions To your LLM provider
Conversation history .context-pilot/ — YAML files Local only
Entity database .context-pilot/shared/entities/ — SQLite Local only
Memories & logs .context-pilot/ — YAML files Local only
Search index Meilisearch on 127.0.0.1 Local only
Search queries Not persisted To Brave Search (if configured)
Text chunks (embeddings) Indexed locally in Meilisearch To Voyage AI (if configured)
Documents (OCR) Cached at ~/.context-pilot/ocr-cache/ To Datalab (if configured)

Compliance

As a local-first tool with no cloud component, traditional compliance frameworks apply differently.

Compatible GDPR

Context Pilot processes no personal data on our behalf. Data sent to LLM providers is governed by your agreement with that provider. We are not a data processor.

N/A SOC 2

SOC 2 applies to cloud services handling customer data. Context Pilot has no cloud service — your data never enters our control.

MIT License Open Source

Permissive MIT license. Use commercially, modify, distribute. No vendor lock-in, no usage restrictions.
