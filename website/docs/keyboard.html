Keyboard Shortcuts

Every keybinding in Context Pilot. No mouse required.

Global Shortcuts

These work from anywhere in the application:

KeyAction
Ctrl+QQuit Context Pilot
Ctrl+LForce redraw / clear screen
Ctrl+NOpen config overlay (provider, model, budget)
Ctrl+HOpen command palette
Ctrl+POpen command palette (alias)
EscapeStop streaming / close overlay / cancel
F12Toggle performance overlay

Input Field

KeyAction
EnterSend message (or continue list item)
Ctrl+EnterInsert newline without sending
← / →Move cursor
Home / EndJump to start / end of line
BackspaceDelete character (or entire paste placeholder)
DeleteDelete character forward
Paste Placeholders

When you paste text, it appears as a colored placeholder showing token and line count. The full content is expanded when you send the message. Backspace over a placeholder removes it entirely.

Sidebar Navigation

KeyAction
↑ / ↓Navigate between panels
dClose selected dynamic panel
PageUp / PageDownPage through dynamic panel list

Panel View (when viewing a panel)

KeyAction
↑ / ↓Scroll panel content
PageUp / PageDownScroll by page
Home / EndJump to top / bottom

Console Panes (Tmux)

When a console panel is selected, most keys are forwarded to the tmux pane:

KeyAction
EnterSend Enter to pane
↑ / ↓Send arrow keys to pane
BackspaceSend Backspace to pane
Ctrl+CSend interrupt to pane
Any characterForwarded to pane
